:code:`nleis.py` extends :code:`impedance.py` with tools for nonlinear
equivalent circuit modeling, EIS analysis, and second-harmonic nonlinear
electrochemical impedance spectroscopy (2nd-NLEIS). The documentation is
organized around the same workflow as the package: install the library,
prepare data, fit EIS and 2nd-NLEIS models, validate results, and inspect the
public API reference.
